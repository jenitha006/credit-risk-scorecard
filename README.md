# Credit Risk Scorecard

A machine learning project to predict loan default probability 
using real-world borrower data.

## Project Overview
Built a credit risk scorecard on 150,000 borrower records to classify 
customers into risk bands — replicating the kind of analysis done by 
risk teams at financial institutions.

## Key Results
- ROC-AUC Score: 0.76
- 4 risk bands with default rates ranging from 1.7% to 17.2%
- Top predictor: Past delinquency history

## Tech Stack
Python | pandas | scikit-learn | imbalanced-learn | matplotlib | seaborn
